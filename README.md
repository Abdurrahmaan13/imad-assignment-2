# imad-assignment-2

#overview 
this app helps users to test their common sense and helps users leran useful real-world shortcuts

#features
The app presents users with simple true false questions 
The app provides instant feedback
The app prvides answer explanations.It allows for the user to undertsand how they went wrong.
The app keeps track oh how much correct answers the user has.
Users can move to the next question by tapping the "next button"
There is a final score displayed at the end of the quiz.

#How it works

When the app starts, it shows the first quiz question on the screen. The user chooses either true or false. After answering, the app checks whether the answer is correct and shows feedback with a short explanation.

The app also keeps track of the user’s score. Once a question is answered, the True and False buttons are disabled so the user cannot answer the same question again. The user must press the next button to continue to the next question.

When all questions are finished, the app opens the `ScoreActivity` screen. It displaya the final score and the total number of questions to that screen, where the user can see their result.
